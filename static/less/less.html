Less入门详解 红字体就是less所表达的css的解释

变量和混合模式less中的变量 用@表示,以及混合(直接拿下面的定义的css 直接来用) 

		  	 	//这里表示2个less变量
		  	 	@bg:#000;
				@test_width:300px;
				.box{
				    width: @test_width;
				    height: @test_width;
				    background: yellow;
				   .borser;//混合，直接拿下面的定义的css 直接来用
				}
				.borser {border: 5px solid #000;}
				
				.box {
					width: 300px;
					height: 300px;
					background: yellow;
					border: 5px solid #000;
					}
				.borser {border: 5px solid #000;}

less的引用 直接写.Class即可 

				//定义个box2 上面的box1完全一样 只是有了些其他不一样的东西 如下
				.box2{
				    .box;//引用上面的css
				    margin-left: 100px;
				}
				
				
					//在这里的box2引用了上面的.box
					.box2 {
					  width: 300px;
					  height: 300px;
					  background: yellow;
					  border: 5px solid #000;
					  margin-left: 100px;
					}

混合-->(可带参数) 定义1个带参数的less，然后后面的class传参数引用他 

	  	 	 //定义1个混合-->(可带参数)
			 .borser_02(@border_width){
			    border:solid yellow @border_width;
			 }
			 //传1个参数进去
			 .test_hunhe{
				    .borser_02(30px);
			  }
				 
				 .test_hunhe {
					 border: solid #ffff00 30px;
				 }
				 
			//混合--->默认带值
			.borser_03(@border_width:10px){
			    border:  solid #00468C @border_width;
			}
			//这里如果不指定的话，默认border宽度为10px
			.test_hunhe_03{
			    .borser_03(2px);
			}
			
			    .test_hunhe_03{
					  border: solid #00468c 2px;
				}

Demo：匹配模式 三角

	  	 	.trangle(top,@w:5px,@c:#ccc){
			    border-width: @w;
			    border-color: transparent transparent @c transparent;
			    border-style: dashed dashed solid dashed;
			}
			.trangle(bottom,@w:5px,@c:#ccc){
			    border-width: @w;
			    border-color:  @c transparent transparent transparent;
			    border-style:  solid  dashed dashed dashed;
			}
			.trangle(left,@w:5px,@c:#ccc){
			    border-width: @w;
			    border-color: transparent @c  transparent transparent;
			    border-style: dashed solid dashed  dashed;
			}
			.trangle(right,@w:5px,@c:#ccc){
			    border-width: @w;
			    border-color: transparent transparent transparent  @c;
			    border-style: dashed dashed  dashed solid;
			}
			.trangle(@_,@w:5px,@c:#ccc){//@_在less表示公共的部分,但是参数必须跟上
			    width: 0;
			    height: 0;
			    overflow: hidden;
			}
			//这句less的class经过上面的渲染就变成下面的了
			.sanjiao{
			   .trangle(bottom,100px) 
			}
			
			.sanjiao {
			  border-width: 100px;
			  border-color: #cccccc transparent transparent transparent;
			  border-style: solid  dashed dashed dashed;
			  width: 0;
			  height: 0;
			  overflow: hidden;
			}

嵌套规则 //&符号表示上一层选择器 超级好用和重要，提升效率的东西 

	  	 	.list {
			  width: 600px;
			  margin: 30px auto;
			  list-style: none;
			  li{margin-top: 20px;background: pink;overflow: hidden;padding-bottom: 10px;}
			  a{float: left;
			     &:hover{color: red;}//&符号表示上一层选择器
			  }
			  span{float: right;}
			}
			
			.list {
			  width: 600px;
			  margin: 30px auto;
			  list-style: none;
			}
			.list li {
			  margin-top: 20px;
			  background: pink;
			  overflow: hidden;
			  padding-bottom: 10px;
			}
			.list a {
			  float: left;
			}
			.list a:hover {
			  color: red;
			}
			.list span {
			  float: right;
			}

@arguments 表示全部的参数 有点和js相似

	  	 	.border_arg(@w:30px,@c:red,@xx:solid){
			    border: @arguments;
			}
			.test_arguments{
			    .border_arg();
			}
			
			.test_arguments {
			  border: 30px #ff0000 solid;
			}

避免被软件编译 ~'' 

	  	 .test_03{
		    width: ~'300px-30px';
		  }
		  
		  	.test_03 {
			  width: 300px-30px;
			}

!important 每一个都会添加上这个 

		  	.test_arguments{
			    .border_arg() !important;
			}
			
			.test_arguments {
				  border: 30px #ff0000 solid !important;
			}
